# 使用说明
## excel表格 说明
* excel表格里面的地址 必须是 以太的0x地址或者ontology的A地址
* amount不要带精度
* 该工具会将20个地址为一组  进行转账，也就是20个地址 为一个交易

## config.json说明

* execute  为false 表示不会发送交易，可以用来测试 去读的excel中的数据是否正确
* execute 为true   才会真正的转账
